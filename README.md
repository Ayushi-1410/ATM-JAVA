Project Description:
 -> This project is a simple ATM Banking System written in Java.
 -> It allows users to:
  1. Create an account
  2. Deposit money
  3. Withdraw money
  4. Transfer money
  5. View account details

It uses Object-Oriented Programming (OOP) concepts like:
   1.Classes
   2.Inheritance
   3.Exception Handling
   4.ArrayList

Installation Instructions
  -> Follow these steps:
      1. Install Java (JDK 8 or above)
      2. Open terminal / command prompt
      3. Save the file as ATMSystem.java
      4. Compile the program:
           -> javac ATMSystem.java
      5. Run the program:
           -> java ATMSystem
           
Usage Instructions

     -> Run the program
     -> You will see a menu:
        1. Create Account
        2. Deposit
        3. Withdraw
        4. Transfer
        5. Display Account
        6. Exit
     -> Enter your choice number
     -> Follow the instructions on screen
